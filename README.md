# 🎵 音乐元数据管理器 - Android应用

## 📋 项目简介

音乐元数据管理器是一款专为Android平台设计的现代化音乐文件管理应用。它能够扫描设备上的音频文件，读取并编辑音乐的元数据信息（如标题、艺术家、专辑等），让您轻松管理音乐库。应用采用Python + Kivy框架开发，具有优秀的Material Design风格界面和流畅的动画效果。

## ✨ 核心功能

### 🔍 智能扫描
- 自动扫描设备Music、Download等常用目录
- 支持多种音频格式：MP3、FLAC、M4A、OGG、WAV、OPUS
- 智能提取音频文件的元数据信息
- 显示歌曲时长、文件名等详细信息

### 📝 元数据编辑
- 编辑歌曲标题、艺术家、专辑名称
- 修改发行年份和音乐流派
- 实时保存修改，永久更新文件元数据
- 友好的表单界面，操作简单直观

### 🔎 快速搜索
- 实时搜索过滤功能
- 支持按标题、艺术家、专辑搜索
- 即时显示搜索结果，无需等待

### 🎨 精美界面
- Material Design设计风格
- 清新的蓝白配色主题
- 圆角卡片式列表布局
- 响应式设计，适配各种屏幕尺寸

### ⚡ 流畅动效
- 列表项淡入动画效果
- 页面切换滑动过渡
- 刷新按钮旋转动画
- Toast提示弹窗效果

## 🏗️ 技术架构

### 开发框架
- **主框架**: Kivy 2.3.0 - Python跨平台GUI框架
- **音频处理**: Mutagen 1.47.0 - 音频元数据读写库
- **语言**: Python 3.x
- **UI语言**: Kivy Language (KV)

### 架构设计
采用MVC架构模式，代码结构清晰：
- **Model层**: `MusicFile`类封装音频文件和元数据
- **View层**: KV文件定义UI布局和样式
- **Controller层**: Screen类处理业务逻辑和用户交互

### 核心组件

#### 1. MusicFile类
封装音频文件的所有元数据操作：
- 自动加载文件元数据
- 解析标题、艺术家、专辑等信息
- 计算并格式化歌曲时长
- 保存修改后的元数据到文件

#### 2. MainScreen主屏幕
应用的主界面，负责：
- 扫描和加载音乐文件列表
- 实时搜索过滤功能
- 列表展示和刷新
- 跳转到详情页面

#### 3. DetailScreen详情屏幕
元数据编辑界面，包含：
- 显示完整的音乐信息
- 可编辑的表单输入框
- 保存和返回功能
- 操作反馈提示

#### 4. MusicListItem列表项组件
自定义列表项，提供：
- 美观的卡片式布局
- 音乐图标和信息展示
- 点击交互响应
- 淡入动画效果

## 📱 用户界面

### 主界面布局
- **顶部导航栏**: 显示应用标题和刷新按钮
- **搜索栏**: 实时搜索音乐
- **音乐列表**: 滚动列表展示所有歌曲，每项包含歌曲图标、标题、艺术家和时长

### 编辑界面布局
- **顶部导航栏**: 返回按钮和页面标题
- **文件信息**: 只读显示文件名和时长
- **编辑表单**: 
  - 标题输入框
  - 艺术家输入框
  - 专辑输入框
  - 年份输入框（仅数字）
  - 流派输入框
- **保存按钮**: 醒目的蓝色按钮

### 交互体验
- 点击列表项滑动进入编辑页面
- 点击返回按钮滑动返回主页
- 输入搜索文字即时过滤结果
- 点击刷新按钮旋转并重新扫描
- 保存成功显示Toast提示

## 🛠️ 技术特性

### 权限管理
- 自动请求存储读写权限
- 兼容Android不同版本的权限系统
- 优雅处理权限被拒绝的情况

### 跨平台支持
- 主要针对Android优化
- 同时支持Windows/Mac/Linux桌面测试
- 自动适配不同平台的文件路径

### 性能优化
- 只扫描一级子目录，避免深度遍历
- 延迟加载UI组件
- 高效的搜索过滤算法
- 内存友好的列表渲染

### 异常处理
- 完善的try-catch错误捕获
- 优雅降级：mutagen未安装时显示基本信息
- 文件访问失败的友好提示
- 元数据保存失败的错误反馈
